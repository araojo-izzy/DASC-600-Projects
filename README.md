# DASC-600-Projects

Python assignments from ODU's DASC 600: Programming for Data Science

## Projects

### PS2 - Down Payment Savings Calculator
A savings calculator that determines how many months it will take to save for a down payment on a home. Takes into account monthly savings rate, investment returns, and semi-annual salary raises.

### PS5 - Penguin Species Data Visualization
A grouped bar chart visualization comparing three penguin species (Adelie, Chinstrap, and Gentoo) across four measurements: bill depth, bill length, flipper length, and body mass. Built using matplotlib and numpy.

## Tools Used
- Python 3
- matplotlib
- numpy
